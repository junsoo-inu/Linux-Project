# 프로젝트 자동화
## 구현 목표 기능
1.	프로젝트 자동 초기화 
프로젝트에 사용 할 언어를 입력 받아 디렉토리를 자동 생성하고 기본 소스 파일( ex. Main.py )을 자동 생성한다.
프로젝트 생성 초기에 git init을 자동으로 실행한다.

2.	gitignore 자동 생성
.gitignore 파일을 자동으로 생성한다.
ex: python: \_\_pychace\_\_ 제외

3.	코드 주석 스캐너
프로젝트 폴더 내 소스 파일 검색 후 소스 파일 내에 키워드(TODO)로 시작하는 주석을 찾아 파일명과 줄 번호를 출력한다

4.	Git 상태 대시보드
현재 git 상태(github 연결여부)를 확인하여 최근 커밋 3개를 확인하고 출력한다.
